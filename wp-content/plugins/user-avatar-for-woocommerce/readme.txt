Aggiunge un semplice avatar al tuo woocommerce riguardo i clienti e al tuo wordpress riguardo l'admin.

== Description ==

Aggiunge l'avatar nella sezione woocommerce nell'area clienti. Il cliente può caricare la foto profilo.

Inoltre Aggiunge un campo di caricamento avatar ai profili utente all'interno della dashboard di WordPress.

Per i siti che tengono gli utenti fuori dalla dashboard si può utilizzare il seguente shortcode "[general-user-avatars]".

Non contiene codice ajax.


= Che cosa bisogna fare? =

Una volta installato il plugin, accedere nel pannello wordpress nel menu "Avatar Woocommerce", ed inserire il flag per attivarlo.
Oltre all'avatar, è' possibile anche attivare la funzione mobile per rendere l'account clienti di woocommerce più navigabile da smartphone (inverte prima il contenuto e dopo il menù)

I clienti possono caricare la loro foto profilo nella sezione "Dettagli Account" nel proprio account.



== Installation ==

Puoi caricare il file zip da Plugin/Aggiungi Nuovo, dal pannello di amministrazione del tuo wordpress 
oppure
Estrai il file zip e rilascia il contenuto nella directory wp-content / plugins / della tua installazione di WordPress, quindi attiva la pagina Plugin da Plugins.
