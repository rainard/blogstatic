A lightweight plugin to display clock widgets on your website.

== Description ==

What Time Is It is a light-weight plugin to let you create and customize your clocks on your website.

Main functions:

* Local clock
* World clock
* Analog clock
* Digital clock
* Current date
* Multiple widgets

For normal users:

* Select clock template (analog/digital)
* Select clock timezone
* Customize clock's color

For developers:

* Customize every elements with CSS
* Full classes for further customization
* Set color for each element
* Set custom clock's background image

== Installation ==

1. Upload the entire `what-time-is-it` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
